# Siege!

### A Deckbuinding, Asymmetrical Card Game


##Deckbuilder

Siege! is a strategy game that revolves around assymetrical play and deckbuilding from a pool of cards. This project will serve as the foundational deckbuilding piece of the game where players can log in and build various decks, mark them as public or private, and save them for the future. 


##Decks

### There are two decks in Siege!: An Invader deck and a Monarch Deck

For each deck, players are required to build a deck with the following rules:
### Invader
- Must choose one Invader leader
- Can add any other Invader cards to their deck providing the sum of deck points is less than the maximum allowed deck points (currently 50) 
- can include up to 3 copies of any card

### Monarch
- Must choose one Monarch to lead the people
- Must choose one Castle to defend
- Can add any other Monarch cards to their deck providing the sum of deck points is less than the maximum allowed deck points (currently 50)
- can include up to 3 copies of any card

## Login
- It is not required to be logged in to create a deck, but a used must be logged in to save a deck.
